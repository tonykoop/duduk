# Visual BOM Brief — Duduk Family

**For:** Graphic production of the visual bill of materials
**Instrument:** Armenian Duduk, A3 root (primary example)
**Date:** 2026-05-08

---

## Layout Spec

- **Format:** Single page, portrait or landscape, PDF-ready
- **Hero image:** Full instrument assembled with reed inserted — angled 3/4 view
- **Color palette:** Warm wood tones; white background; black text
- **Style:** Technical documentation — no decorative borders

## Table Structure

| Col | Content |
|---|---|
| # | Part ID (DUD-A3-001 etc.) |
| Image | Thumbnail photo or sketch of part |
| Description | Part name + brief description |
| Qty | Quantity |
| Unit | ea / oz / set |
| Material | Primary material |
| Est. Cost | USD each |
| Source | Category (specialty lumber, music supplier, hardware) |

## Part Thumbnails Needed

1. **Body tube (DUD-A3-001):** Photo/sketch of a cylindrical wood tube, showing bore end and bell end. Dimensions labeled: 0.748″ OD, 0.472″ bore, ~13.25″ body.
2. **Double reed (DUD-A3-002):** Photo of an Arundo donax duduk reed showing the wide flat tip, tapered shank, and bridle band. Annotate: tip, bridle, shank.
3. **Bridle (DUD-A3-003):** Close-up of bridle band on reed — thread-wound or rubber band style.
4. **Bore oil (DUD-A3-004):** Small oil bottle icon or photo.
5. **Exterior finish (DUD-A3-005):** Tung oil bottle.
6. **Reed insert wrap (DUD-A3-006):** Roll of PTFE tape.

## Notes

- All images must show actual parts, not mood boards.
- Mark reed image with note: "Reed source — Arundo donax, specialty supplier required"
- Add footer: "All costs estimated as of 2026-05-08. Verify before ordering."
- Add cultural note watermark or sidebar: "The duduk is a UNESCO Intangible Cultural Heritage instrument. Reed-making is a traditional craft skill not captured in this BOM."
